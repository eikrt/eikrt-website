Mechaball

Github link

If you haven't played Breakout or it's huge offspring, you have missed a slice of history. Breakout has numerous follow-ups, Arkanoid being the most famous of them. My favourite iteration of the concept is DX-Ball

My take on "ball and paddle" genre is Mechaball. Mechaball is written in Python and uses the curses library. The program is pretty simple, but needs some elegance here and there. Ball reflection mechanic is also pretty dull.

You can clone the repository or ssh mechaball@eikrt.com . Hope you enjoy my little game!

Heartlands

Github link
App link

For a long time I have wanted to create a world generation software. I have always enjoyed games like Dwarf Fortress or Minecraft where events take place on a procedurally generated world.

My world generation currently has height map created with perlin noise . Currently there are no more features.

The project is intended to be modular, so each part can be modified independently. Someday I will create different choices of frontends, maybe C/C++ with Qt or something with curses. Mobile app would also be a fun project.
